# Data Centers and Climate Impact

This project visualizes data center locations and estimates their potential impacts on electricity demand, carbon emissions, and local utility costs across the United States, and with one particular focus on Ashburn, Virginia. By using public facility datasets, the visualizations help compare states and facilities and demonstrates how concentrated data center growth can go through energy systems and households.

Installation & setup
- Clone or download the repository.
- From the project root, enter these commands in the terminal to start the webpage:

```bash
# from the project folder
python3 -m http.server 9000
# then open http://localhost:9000 in your browser
```

No other build tools or external dependencies are required, the site is static and uses D3 via CDN.


Lessons learned
- The goal is to show the human and infrastructure trade-offs that follow concentrated data center growth, energy, electricity, costs, and emissions — so readers can see what matters.
- The visualizations are designed to support any journalists, planners, and community members in asking better questions about proposed facilities and grid impacts.
- The project aims to make the consequences of siting and scale visible so decision-makers can consider alternatives.
